# Data

This directory contains reproducible synthetic sample data used by examples, tests, validation reports, and the Streamlit dashboard.

- `sample_market_events.csv` is produced by `examples/generate_validation_report.py`.
- `sample_execution_schedule.csv` is produced by the same validation workflow.

The sample data is synthetic. It is intended for engine validation and demonstration of market microstructure workflows, not as evidence of live trading performance.
